# Task Manager Project 
Консольное приложение для управления задачами.

## Стек технологий
- Java 21 (Core, Collections, Исключения)
- Maven
- JUnit 5
- Git
- PostgreSQL
- JDBC
- SLF4J + Logback

## Функционал
- Добавление задачи — описание проходит валидацию
- Удаление задачи по ID
- Изменение статуса задачи: выполнена / не выполнена / заброшена
- Просмотр всех задач
- Фильтрация задач по статусу
- Сохранение и загрузка задач 
- ID присваивается автоматически, у каждой задачи есть дата создания.

## Хранение данных
- Реализован интерфейс Storage для работы с разными способами хранения
- FileStorage - сохраняет задачи в текстовый файл (для безопасности используется также временный файл)
- DatabaseStorage - сохраняет задачи в базу данных с использованием commit/rollback

## Архитектура
- Проект состоит из частей: ui, storage, model, manager, commands
- Обработка действий пользователя реализована при помощи паттерна Command

## Тестирование
- 10+ unit-тестов для проверки бизнес-логики
- Протестированы основные сценарии: добавление, удаление, изменение статуса
- Проверка некорректных данных (null, пустые строки, пробелы, переносы строк, невалидные символы)
- Тесты написаны на JUnit 6
