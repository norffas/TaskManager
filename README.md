# Task Manager Project 
Консольное приложение для управления задачами.

## Стек технологий
- Java 17 (Core, Collections, Исключения)
- Maven
- JUnit 5
- Git

## Функционал
- Добавление задачи — описание автоматически очищается от пробелов по краям
- Удаление задачи по ID
- Изменение статуса задачи: выполнена / не выполнена
- Просмотр всех задач
- Фильтрация задач по статусу
ID присваивается автоматически, у каждой задачи есть дата создания.

## Тестирование
- 10+ unit-тестов для проверки бизнес-логики
- Протестированы основные сценарии: добавление, удаление, изменение статуса
- Проверка некорректных данных (null, пустые строки, пробелы, переносы строк)
- Тесты написаны на JUnit 5
